# Pet Friendli

Marca emocional que transforma el vínculo entre una persona y su mascota en algo tangible que puede conservar, llevar consigo y compartir con otros.

## Stack
- React + Vite + Tailwind CSS
- Backend futuro: Flask + PostgreSQL + SQLAlchemy

## Estructura
pet-friendli/
├── frontend/
│   └── src/
│       ├── components/
│       ├── layouts/
│       ├── pages/
│       ├── constants/
│       └── styles/
└── CONTEXTO.md
## Arrancar el proyecto
cd frontend
npm install
npm run dev
